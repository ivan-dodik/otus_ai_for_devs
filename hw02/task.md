# Промпты

## Первоначальный промпт

ЗАДАНИЕ: выполнить лабораторную работу "Генерация простого приложения в IDE"

ЦЕЛЬ: сгенерировать и собрать простое приложение «Мини-анкета», состоящее из backend-части и небольшого frontend-интерфейса.

ЗАДАЧИ:
 * создать минимальное full-stack приложение (backend + frontend) с помощью AI-ассистента в IDE;
 * настроить API для получения конфигурации анкеты и отправки ответов;
 * научиться просить ассистента структурировать архитектуру и генерировать отдельные файлы;
 * развернуть фронтенд, подключенный к backend API;
 * реализовать хранение пользовательских ответов в памяти backend-приложения;
 * проверить работу приложения локально.


ТРЕБОВАНИЯ:

Backend предоставляет два API:
 * GET /questions — возвращает список вопросов анкеты (жёстко заданных, 3–5 шт.).
 * POST /answers — принимает ответы пользователя и сохраняет их в памяти (например, в slice/array/map).

Frontend:
 * загружает вопросы с backend;
 * отображает их в HTML/JS интерфейсе;
 * отправляет заполненные ответы через POST /answers;
 * показывает пользователю сообщение «Спасибо!» после отправки.

Технологии:
 * Backend: Python
 * Frontend: чистый HTML+JS

УЧЕБНАЯ ЦЕЛЬ

В данном задании тренируются навыки:
 * интеграции IDE и AI-ассистентов в разработку;
 * генерации проектной структуры через ассистента;
 * создания простых REST API (GET/POST);
 *организации фронтенда, взаимодействующего с API;
 *отладки кода, возникших ошибок и корректировки моделей;
 * запуска full-stack приложения локально.

ФОРМАТ СДАЧИ
 * файлы проекта;
 * короткую инструкцию по запуску
 * скриншоты работы
 * использованные промты

Сохраняйте все промты, использованные в процессе!
