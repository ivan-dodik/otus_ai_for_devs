Этот файл документирует все промпты AI-ассистента, использованные для создания приложения "Мини-анкета".

## Промпт 1: Создание структуры проекта и кода backend/frontend

```
Создай минимальное full-stack приложение "Мини-анкета". 

Backend (Python + Flask):
- Создай файл backend/app.py с приложением Flask
- Реализуй два API endpoint:
  1. GET /questions - возвращает JSON со списком из 3-5 вопросов анкеты (например, имя, возраст, любимый язык программирования)
  2. POST /answers - принимает JSON с ответами пользователя и сохраняет их в памяти (в список или словарь)
- Добавь CORS заголовки для возможности запросов с frontend
- Создай backend/requirements.txt с зависимостями (flask, flask-cors)

Frontend (HTML + JavaScript):
- Создай frontend/index.html с формой для ответов на вопросы
- Создай frontend/script.js с логикой:
  1. При загрузке страницы делает GET запрос на /questions и динамически создаёт форму
  2. Отправляет заполненные ответы через POST /answers
  3. Показывает сообщение "Спасибо!" после успешной отправки
- Frontend должен быть подключен к backend API (предполагается, что backend запущен на http://localhost:5000)

Создай всю структуру папок и файлов. Код должен быть готов к запуску.
```

**Созданные файлы:**
- `backend/app.py`
- `backend/requirements.txt`
- `frontend/index.html`
- `frontend/script.js`

## Промпт 6: Исправление ошибки установки зависимостей uv

При проверке запуска в чистой системе (без заранее созданного venv) возникла ошибка:
```
error: No virtual environment found; run `uv venv` to create an environment, or pass `--system` to install into a non-virtual environment
```

**Причина:** Команда `uv pip install -e backend/` требует существования виртуального окружения, но не создает его автоматически.

**Решение:** Использовать `uv sync` в директории backend для автоматического создания venv и установки зависимостей, затем использовать `uv run` для запуска backend.

**Измененные файлы:**
- `start.sh`:
  - Замена `uv pip install -e backend/` на `(cd backend && uv sync)`
  - Замена `python3 app.py` на `uv run python3 app.py`
- `start.py`:
  - Замена `uv pip install -e backend/` на `uv sync` с `cwd='backend'`

**Обновленные команды:**
```bash
# Установка зависимостей (теперь создает venv автоматически)
(cd backend && uv sync)

# Запуск приложения
uv run python backend/app.py

# Запуск тестов
uv run python test_api.py
```

**Созданные файлы:**
- `backend/pyproject.toml` (вместо requirements.txt)
- Обновленные версии всех файлов с русским текстом
- `CHANGES.md` - отчет о внесенных изменениях
