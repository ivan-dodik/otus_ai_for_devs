# Project Brief

## Overview
Betaflight is flight controller software (firmware) used to fly multi-rotor craft and fixed wing craft. It focuses on flight performance, leading-edge feature additions, and wide target support.

## Core Requirements
- High-performance flight control firmware for multi-rotor and fixed-wing aircraft
- Support for a wide range of hardware targets (STM32 F4, G4, F7, H7, C5, H5, N6 processors, and others)
- Rich feature set including OSD, Blackbox logging, telemetry, LED strip support, and more
- Configurable via the Betaflight App (progressive web app)

## Key Goals
- Flight performance optimization
- Leading-edge feature additions
- Wide target support across multiple MCU families
- Open source community-driven development
- Regular release cadence (two major releases per year)

## Versioning
- Format: `YYYY.M.PATCH` (e.g., `2025.12.1`)
- Release cadence: Two major releases per year (June and December)
- Development phases: Alpha → Beta (1 month) → Release Candidate (1 month) → Release
